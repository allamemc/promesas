API CON PROMESAS

API CON ASYNC AWAIT

Async declara una función asincrónica que devuelve Promesas, y await se utiliza para esperar que las Promesas se resuelvan de manera síncrona, lo que simplifica el manejo de código asincrónico y lo hace parecer más similar a código síncrono, lo que mejora la legibilidad y el mantenimiento del código.
